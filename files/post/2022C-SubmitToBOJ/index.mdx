연습문제 내에 각 문제로 링크를 걸어 놓았습니다. 들어가면 아래와 같은 화면을 볼 수 있습니다.

![Untitled](/post/2022C-submitToBOJ/Untitled.png)

시간 제한 : 작성된 프로그램이 실행되는 시간이 이 기준을 넘어가면 안 됩니다. 아직은 신경쓰지 않으셔도 괜찮습니다.

메모리 제한 : 작성된 프로그램이 사용하는 메모리가 이 기준을 넘어가면 안 됩니다. 역시 아직은 신경쓰지 않으셔도 괜찮습니다.

예제 입력/출력 : 문제를 제출하면 채점 서버에서는 작성하신 프로그램을 실행시킨 후, 예제 입력과 **같은 형식**으로 입력을 줍니다. 그 결과로 예제 출력과 **같은 형식**으로 답이 올바르게 나온다면, <span style={{color:'green'}}>**맞았습니다**</span>를 받습니다. 만약 형식이 다르거나, 답이 올바르지 않다면 <span style={{color:'red'}}>**틀렸습니다**</span>를 받습니다.

여기에서 형식이란, 띄어쓰기의 유무, 줄바꿈의 유무 등이 포함됩니다.

제출은 문제 상단의 탭에서 제출을 눌러주세요.

![Untitled](/post/2022C-submitToBOJ/Untitled%201.png)

언어 : C언어로 코드를 작성했으므로 C99를 선택해주세요.

소스 코드 공개 : 제가 코드를 확인할 수 있도록 꼭 공개로 설정해주세요.

소스 코드 : 작성하신 소스코드 전체를 복사해서 붙여넣으시면 됩니다.

세 가지 모두 확인하신 후에 아래 제출 버튼을 누르면 채점 서버에서 채점 후에 결과를 알 수 있습니다.
