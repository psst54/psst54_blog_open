VS에서 C언어로 프로그램을 작성하기 위해서는 새로운 프로젝트를 생성해야 합니다.

![Untitled](/post/2022C-newProject/Untitled.png)

이전 글을 따라서 VS 설치를 마치셨다면, 위와 같은 창을 볼 수 있습니다. 오른쪽 버튼 중에서 <Highlight>**새 프로젝트 만들기**</Highlight> 버튼을 클릭해주세요.

![Untitled](/post/2022C-newProject/Untitled%201.png)

<Highlight>**빈 프로젝트**</Highlight>가 선택된 채로 <Highlight>
  **다음**
</Highlight> 버튼을 눌러 주세요

![Untitled](/post/2022C-newProject/Untitled%202.png)

원하는 프로젝트 이름을 입력하고, 위치를 선택해주세요. 프로젝트 이름은 영어로 짓는 것을 권장드립니다. 그 다음에 <Highlight>**만들기**</Highlight> 버튼을 눌러주세요.

![Untitled](/post/2022C-newProject/Untitled%203.png)

이런 창이 켜진다면 성공적으로 프로젝트가 만들어진 것입니다🎉🎉

프로젝트가 만들어졌다면, 코드를 작성하기 위한 소스 파일을 만들어야 합니다.

![Untitled](/post/2022C-newProject/Untitled%204.png)

솔루션 탐색기 창에서 <Highlight>**소스 파일**</Highlight> - <Highlight>**추가**</Highlight> - <Highlight>**새항목**</Highlight>을 차례로 클릭해주세요.

![Untitled](/post/2022C-newProject/Untitled%205.png)

C++파일(C파일은 없으니 찾지 않으셔도 됩니다!)을 선택하고, 이름을 입력할 때 (파일이름).c로 입력해주세요.

기본적으로 (파일이름).cpp로 설정되어 있지만, 지우고 .c 파일로 만들어 주세요!

![Untitled](/post/2022C-newProject/Untitled%206.png)

이렇게 소스 파일 아래에 (입력한 파일이름).c 파일이 생성되면 소스 파일까지 만들어진 것입니다🎉🎉 다음 글에서는 이렇게 만들어진 파일에 코드를 작성하고, 코드를 실행해 보는 시간을 가지겠습니다. 다음 글에서 뵙겠습니다!

추가로, VS를 종료하고 다음번에 실행하면 아래와 같은 창이 표시됩니다. 위에서 입력한 프로젝트 이름을 더블클릭해서 들어오시면 프로젝트를 다시 열 수 있습니다.
