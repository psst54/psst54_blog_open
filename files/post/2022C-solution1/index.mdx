## 1. Hello World

[https://www.acmicpc.net/problem/2557](https://www.acmicpc.net/problem/2557)

```c
#include <stdio.h>
int main(void)
{
		printf("Hello World!");
		return 0;
}
```

`Hello World!`의 철자는 틀리면 안 됩니다.

## 2. **We love kriii**

[https://www.acmicpc.net/problem/10718](https://www.acmicpc.net/problem/10718)

```c
#include <stdio.h>
int main(void)
{
    printf("강한친구 대한육군\n강한친구 대한육군");
    return 0;
}
```

`강한친구 대한육군`을 한 번 출력하고 `\n`을 출력해서 줄을 바꿔 주어야 합니다.

## 3. **고양이**

[https://www.acmicpc.net/problem/10171](https://www.acmicpc.net/problem/10171)

```c
#include <stdio.h>

int main(void)
{
	printf("\\    /\\\n");
	printf(" )  ( ')\n");
	printf("(  /  )\n");
	printf(" \\(__)|\n");
	return 0;
}
```

`\`를 출력하고 싶으면 `printf(”\\”)`처럼 출력하도록 합니다.

## 4. 개

[https://www.acmicpc.net/problem/10172](https://www.acmicpc.net/problem/10172)

```c
#include <stdio.h>

int main(void)
{
	printf("|\\_/|\n");
	printf("|q p|   /}\n");
	printf("( 0 )\"\"\"\\\n");
	printf("|\"^\"`    |\n");
	printf("||_/=\\\\__|\n");
	return 0;
}
```

`\`를 출력하고 싶으면 `printf(”\\”)`처럼 출력하도록 합니다.

`“`를 출력하고 싶으면 `printf(”\"”)`처럼 출력하도록 합니다.
