### 경험적 실질론

경험된 세계가 현실적으로 있다

### 세계에 대한 인식 조건

1. 감성 형식 : 수용, 수동성
   - 공간은 외부 지각 또는 외부 직관이 가능한 형식
   - 시간은 모든 직관, 곧 내부 직관과 외부 직관이 가능한 필수 조건
     - 시간은 내부 현상이 가능한 직접적 조건이고 외부 현상이 가능한 간접적 조건
   - 외부 대상을 일정한 공간 관계에 있는 것으로 정리해서 받아들이고, 동시에 표상되는 대상을 동시에 시간 순서에 따라 직관; 표상을 수용하는 능력
   - 감각 지각 표상을 수용하는 능력
2. 오성 형식 : 자발, 능동성
   - 오성의 형식은 12범주
     - 양(단일성, 다수성, 전체성)
     - 질(현실성, 부정성, 제한성)
     - 관계(실체-속성, 원인-결과, 상호작용)
     - 양상(가능, 실제, 필연)
   - 감성을 통해 주어진 다양한 자료를 결합하고 통일하는 방식
   - 순수 오성 개념(범주, 규칙)을 자발적으로 만듦
   - 인간 공동체가 만들어냄(개인이 만든 것이 아님)
3. 통각
   - 내가 생각한다는 자기의식. 범주적 통일의 근거.
   - 모든 표상에 수반할 수밖에 없음. 개념을 판단할 때 늘 나의 의식이라는 것이 동반됨
   - 초월적 자아
     - 경험적 자아와 초월적 자아 구별
     - 경험적 자아 : 현상으로서 나
     - 초월적 자아 : 현상으로서 나를 의식하는 근원적인 나
   - 종합하고 통일하는 기능
   - 객체 또는 대상 인식의 바탕

### 일반 논리학의 판단 분류표

- 양 : quantity
  1. 단칭 판단 : a는 p다
  2. 특칭 판단 : 어떤 S는 p다
  3. 전칭 판단 : 모든 S는 P다
- 질 : quality
  1. 긍정 판단 : S는 P다
  2. 부정 판단 : S는 P가 아니다
  3. 무한 판단 : S는 비-P다
     - 비-P인 것들의 집합이 무한히 많아질 수 있다
- 관계 : relation
  1. 정언 판단 : S는 P다
     - 실체 속성 개념이 깔려 있음
     - 특정 시기 동안 지속하는 존재를 실체라고 이야기함. 실체는 속성을 가짐. 실체는 특정 기간 동안 있으면서 속성이 바뀜.
     - 실체에 대한 개념을 형성했기 때문에 정언판단이 가능함
  2. 가언 판단 : p이면 q
  3. 선언 판단 : p이거나 q
- 양상 modality
  1. 가능 판단 : S는 P일 수 있다; S가 P인 것은 가능하다
     - 가능 불가능에 대한 개념을 우리가 가지고 있다
  2. 실제 판단 : S는 실제로 P다; S가 P인 것은 참이지만 필연적인 것은 아니다
     - 시간 공간적으로 어떤 위치를 점하고 있다
  3. 필연 판단 : S는 P이어야 한다; S가 P인 것은 필연적이다
     - 우리의 오성은 필연적인 것과 아닌 것을 구분할 수 있는 능력이 있다

⇒ 이런 개념을 우리가 만들어냈기 때문에 단칭 특칭 전칭 판단이 가능함. 이 판단의 가능 근거를 검토하고 연합하는 것이 초월주의

### 초월 논리학의 범주(개념) 분류표

- 양 ; 수학적 범주; 지각 표상에 적용
  1. 단일성 : 하나;
     - 태양을 단일한 것으로 헤아림
  2. 다수성 : 여럿
     - 행성을 여러 개로 수치화
  3. 전체성 : 모두
     - 태양계의 행성이 모두 8개라고 판단
- 질 : 수학적 범주; 지각 표상에 적용
  1. 현실성 : 감각 내용이 대상에 있음
     - 다이아몬드는 단단하다
     - 이런 것에 입각해 S는 P다 등을 판단할 수 있음
  2. 부정성 : 감각 내용이 대상에 없음
     - 다이아몬드는 유기체가 아니다
  3. 제한성 : 긍정과 부정을 동시에 공유
     - 다이아몬드는 비유기체이다
- 관계 : 역학적 범주; 물리적 대상, 사건에 적용
  1. 실체와 우연적 속성 : 실체는 지속, 속성이 바뀜
  2. 원인과 결과 : 발생하는 것의 필연적 관계
     - 사건에 대해 원인을 찾도록 하는 사고 방식이 우리에게 있음
  3. 상호성 : 모든 실체는 상호작용하는 것
- 양상 : 역학적 범주; 사태나 사실에 적용
  1. 가능-불가능 : 발사체의 포물선 운동은 가능하다
  2. 실제로 있음-부재 : 중력의 영향은 실제로 있다
  3. 필연-우연성 : 지구상 물체가 중력의 영향을 받는다는 것은 필연적이다

### 선험적 종합 원칙 : 수학 원칙과 역학 원칙

- 직관의 공리
  - 모든 현상은 직관에서 외연적 크기다
  - 외연적 크기의 도식은 수
  - 현상을 하나, 여럿, 모두로 판단해서 인식
  - 선험적 종합 판단; 경험에서 유래하지 않은 오성으로 판단한 선험적 지식
- 지각의 예상
  - 모든 현상에서 감각과 감의 대상에 대응하는 현실적인 것은 내포적 크기나 정도를 가짐
  - 감각의 대상으로 주어진 것, 질료(감각 내용)를 ‘무엇’으로 인식
  - ex) 풀은 푸른 색이다, 물이 차갑다
- 경험의 유추
  - 경험의 대상은 현상. 이 현상들의 관계를 규정하는 걳이 경험의 유추
  - 경험은 지각 내용들의 필연 관계를 표상해야만 가능하다
  - 현상(지각 내용들)은 고정되거나, 잇따르거나, 동시에 있음
- 경험적 사유 일반의 요청
  - 경험의 형식적 조건(직관과 개념)고=과 합치하는 것은 가능함
  - 경험의 질료적 조건(감각)과 관련된 것은 실제로 있음
  - 감각되지 않은 것은 실제로 있지 않음
  - 실제로 있는 것과 관련해 경험의 보편적 조건에 따라 결정되는 것은 필연적임

### 경험의 유추

1. 첫째 유추 : 실체 개념의 객관적 사용 원칙

   > A판 : 모든 현상은 고정불변하는 실체와 변하는 우연적인 것을 포함한다

   > B판 : 현상들의 모든 변화에도 실체는 고정불변하며, 자연에서 실체의 양은 증가하지도 감소하지도 않는다

   실체는 시간 표상과 공간 표상이 성립할 수 있는 기본 개념이자 지각들의 모든 종합적 통일, 경험을 가능하게 하는 조건

2. 둘째 유추 : 인과 개념의 객관적 사용 원칙

   > A판: 발생하는, 곧 존재하기 시작한 모든 것은 규칙에 따라 뒤따르는 어떤 것을 선제한다

   > B판: 모든 변화는 인과 관계의 법칙에 따라 일어난다

   어떤 사태가 발생하면, 발생한 사태의 조건인 다른 어떤 사태가 필연적으로 선행함; 선행하는 사태는 원인, 뒤따른 사태는 결과; 주어진 지각 내용을 원인과 결과의 개념으로 종합함으로써 인과 판단, 인과 지식을 얻음

3. 셋째 유추 : 상호성 개념의 객관적 사용 원칙

   > A판: 모든 실체는 공존하는 한, 전면적 관계를 맺는다

   > B판: 모든 실체는 공간 속에 공존할 수 있는 한, 전면적 상호관계를 맺는다

   한 사물에 대한 지각이 다른 사물에 대한 지각에 뒤따르고, 후자에 대한 지각이 전자에 대한 지각에 상호적으로 뒤따르면, 두 사물은 공존

## 코페르니쿠스식 전환과 초월 논증

### 코페르니쿠스식 전환

코페르니쿠스의 지동설과 유사한 인식론의 혁명

인식 조건이 대상이 가능한 조건이다

인식하는 것과 인식되는 것이 분리되지 않음

- 인식이 대상에 준거함으로써 가능하면, 우리는 결코 필연성과 보편성을 가진 지식을 획득할 수 없음
- 대상이 우리의 의식에 준거한다고 가정하면, 대상에 대해 선험 지식, 곧 보편성과 필연성을 가진 지식을 얻을 수 있음

태어날 때부터 모든 사물을 빨간색으로 보도록 시각 능력이 고정된 사람의 예

1. 모든 것이 빨갛다
2. 사물들의 색이 다양하지만, 주관적 조건 때문에 빨갛게 보인다

- 칸트는 2를 의도함

우리가 안 지식이 사물 자체에 대한 지식이다는 주장 → 합리론자의 독단성에 빠지는 주장.

- 우리 앎은 현상계에 국한됨.
- 현상계 너머에 대해서는 추론할 수 있지만, 관념적인 것임.

### 선험적 종합 판단

경험에 의존하지 않고, 경험과 독립적으로 알려지기 때문에 필연성과 보편성을 가지면서도 지식을 확장하는 지식

(1) 2+5=7
(2) 두 점을 잇는 최단 거리는 직선이다.
(3) 발생하는 모든 것에는 원인이 있다,
(4) 자연계에서 물질의 양은 언제나 같다.
(5) 물질계 안에서 일어나는 운동에서 작용과 반작용의 운동량은 같다.

모든 선험적 지식은 개념적 지식

선험적 종합 판단은 개념들을 모든 경험에 ‘앞서’ 결합하는 판단; 우리는 어떤 일이 발생하면 거기에 원인이 있음을 선험적으로 안다; 인과성 개념을 적용해서 발생하는 현상을 판단하게 되어 있기 때문

현상계를 객관적으로 알 수 있다는 점에서 경험적 실재론, 사물 자체를 알 수 없고, 사유할 수 있을 뿐이라는 점에서 초월적 관념론

## 경험적 실재론과 초월적 관념론

### 경험적 실재론

- 우리의 인식 조건에 따라 알 수 있는 현상계에 대해 객관적 인식이 가능함
- 인식의 조건이 대상의 가능 조건이므로, 현상계(자연계, 물리계)가 객관적으로 실존하느냐는 문제는 애초에 제기할 수 없음
- 세계는 우리가 경험, 인식하는 대로 객관적으로 실존함
- 합리론자의 유심론이나 경험론자의 관념론과 다름

### 초월적 관념론

- 인간은 현상계 너머에 그 자체로 있는 초월적 현실, 사물 자체를 생각할 수는 있을 뿐, 알 수 없음
- 우리가 아는 세계는 경험할 수 있는 현상계이고, 현상계에 대한 우리의 지식이 사물 자체에 대한 것인지를 우리는 결코 확인할 수 없음
- 초월적 대상, 사물 자체에 대해 사유할 수 있고, 이념으로서 존재한다고 추론할 따름

### 두 이론은 양립할 수 있는가?

- 칸트는 정신적 실체와 물질적 실체의 이원성을 주장하지 않음
- 칸트의 자아는 세계와 분리된 사고하는 실체가 아님
- 자아는 세계와 연결되어 있으며, 현상을 종합하는 활동으로 자신을 세계 속에 실현함; 자신에게 주어진 감각 내용, 현상을 선험적 범주로 종합해 판단함으로써 세계의 현실성을 확인
- **경험적 실재론**은 우리가 인식한 현상계가 객관적으로 존재한다는 **현상 존재론**, **초월적 관념론**은 \***\*인간 지식의 본성과 한계를 보여주는 **형이상학 이론\*\*
