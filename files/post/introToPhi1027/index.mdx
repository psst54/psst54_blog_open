# 칸트의 비판 철학 / 인식론

칸트를 기점으로 현대 철학이 시작되었다.

흄의 영향을 받아 여전히 합리론자이긴 하나 합리론적 독단성을 벗었다.

- 우리가 세계를 그대로 반영하는 것이 아니라, 어떤 점에서 세계를 구성한다는 관점이 시작된다.
- 구성 : 우리가 개개의 정신인데, 그 개개의 정신들이 공통된 형식, 조건을 갖추고 있다.
- 공통적인 인식 조건에 따라서 세계에 대해서 판단하면 그것은 객관적이고 보편적이고 필연적인 지식으로 성립 가능하다고 주장한다.
- 하지만 그렇게 판단한 지식이 사물 자체와 일치하는지는 대조할 수 없다.

## 18세기 계몽 운동과 칸트 비판 철학의 과제

### 18세기 계몽주의

자본주의, 현대 민주주의(자유주의 정치 사상)의 시작

칸트는 프랑스 계몽주의에 영향을 받아 비판 철학을 정립함.

- 프랑스 계몽주의 : 합리론, 경험론, 낭만주의, 유물론과 무신론, 사회주의와 낙관론의 특징을 다양하게 보임.
- 프랑스 계몽주의는 17세기 합리론자들과 달리 모든 것을 이성으로 알 수 있다고 말하지 않음. 자연계의 법칙을 알아내어 인간의 삶을 더 낫게 만들 수 있다는 낙관론이 지배함.

17세기 과학 혁명의 결과로 얻은 지식과 철학에서 인간 중심주의를 대중에게 전파

### 칸트 비판 철학의 과제

계몽 : 인간이 다른 사람의 의견에 불종하는 상황에서 해방돼서 자신의 이성을 자유롭게 독자적으로 사용하는 것

⇒ 자신의 이성을 스스로 사용하지 못하는 미성숙에서 벗어나는 것.

> 과감히 분별하라

> 너 자신의 이성을 사용할 용기를 내라

우리는 이성으로 사물 세계에 대한 지식을 얻고 추론을 하는데, 이성 자체가 그럴 자격이 있는가?

- 인간이 무엇인가?; 우리는 무엇을 알 수 있는가?, 우리는 무엇을 희망해도 좋은가?, 우리는 무엇을 행해야 하는가?에 대한 비판서 ⇒ 순수 이성 비판, 실천 이성 비판, 판단력 비판

- **순수 이성 비판** : 인간이란 무엇인가? 우리는 무엇을 알 수 있는가?
  - 이론 이성을 비판적으로 검토하면서 인간이 어떻게 세계를 객관적으로 인식할 수 있는지, 인식의 범위와 한계를 정함
  - 이론 이성은 감성, 오성, 이성으로 세분.
  - 초월적 요소론과 초월적 방법론
  - 이성이 스스로 자신을 비판하는 것, 곧 순수 이성이 자신의 한계를 규정함
  - 흄의 회의론에 맞서 세계를 객관적으로 인식할 수 있는냐는 문제에 답을 찾으려 시도
- **실천 이성 비판** : 우리는 무엇을 희망해도 좋은가?
  - 종교와 신학에 대한 논의
  - 순수 실천 이성에서 유래하는 도덕 의식이 자유를 요청함
  - 자유는 순수 실천 이성의 사실
    > 숙고하면 할수록 점점 더 새롭고 더 경탄과 외경으로 마음을 채우는 두 가지는 내 위에 별이 빛나는 하늘과 내 안에 도덕법칙이다.
  - 도덕 행위가 가능하기 위한 조건 : 자유 의지, 영혼의 불멸, 신의 실존을 요청 ← 우리의 경험의 한계를 넘어섬. 다만 우리의 도덕이 가능하기 위해서 요청되는 것으로 있음.
- **판단력 비판** : 우리는 무엇을 행해야 하는가?
  - 도덕 형이상학의 가능성 탐구

### 평가

근대 철학에서 중세 사고방식의 흔적을 걷어내고 현대 철학으로 나아가는 길을 열었음

## 비판 철학으로서 인식론

### 선험과 초월

선험(a priori) : 경험에 앞서서 알려짐

초월(transcendental) : 경험에 의존하지 않고 알려짐, 경험을 가능하게 만듦, 경험의 한계를 넘어서 경험을 가능하게 만드는 조건

⇒ 따라서 칸트 철학의 주요 과제는 선언적 또는 초월적 인식의 본성과 한계, 타당성 또는 유효함을 증명하는 것.

선험적이면서 단순히 분석적이지 않은 명제로 구성된 학문 : 수학과 형이상학

- 분석 명제 : 주어 개념 속에 술어 개념이 포함되어 있음.
  - ex) 삼각형은 각이 3개 있다.
- 종합 명제 : 주어 개념에 뭔가를 덧붙이는 명제
- 선험적 종합 명제 : 경험에 앞서서, 경험에 의존하지 않고 알려지지만 새로운 정보를 주는 것
- 두 학문 사이에 물리학이 있음 : 실체의 지속서오가 인과율같은 선험적 종합 원칙을 전제함.

⇒ 수학과 형이상학에 대한 합리론과 경험론의 주장을 각각 비판하고, 새로운 견해를 제안함.

### 초월 논증

- 데카르트, 스피노자, 라이프니츠는 수학과 형이상학을 선험적 학문이라고 생각함.
  - 라이프니츠는 모든 선험적 명제는 분석적이라고 주장함.
  - 칸트는 “7 + 5 = 12”같은 산술 명제와 “두 직선은 공간을 에워쌀 수 없다”같은 기하학의 명제가 종합적이라고 확신함 : 두 직선 속에 공간을 에워쌀 수 없음이 포함되어 있지 않다.
- 흄은 경험론자로서 산술 명제가 선험적이고 분석적이며, 기하학의 명제는 종합적이지만 경험적이라고 여겼던 것 같다.
  - 인과율과 실체의 지속성에 대한 믿음을 습관과 연상의 산물로 여기고 형이상학에 대해 회의론적 견해를 내놓음.
  - 칸트는 전통 형이상학에 대한 흄의 회의론적 결론(세계 전체, 사물 전체, 신, 영혼 불멸 등에 대해서 알 수 없다)을 받아들이지만, 흄의 회의론에 맞서 인과율이나 실체가 지속한다는 명제는 증명이 필요하고 증명될 수 있는 선험적 지식이라고 생각함.

어떤 지식 명제가 선험적임을 증명하는 논증을 ‘초월 논증(transcendental argument)’이라고 부름

- 가능한 경험의 한계 안에서 특정 지식이 성립할 수 있다는 것을 보여주는 것
- 순수 이성 비판 전체가 초월 논증의 절차를 밟고 있음
- 경험이 가능하기 위한 인식 조건을 증명하는 것
- 선험적 종합 인식이 가능한 근거를 제시하는 것

### 동기

수학과 형이상학은 모두 선험적 학문임 : 경험과 독립적으로 알려지고, 필연적이라고 주장하는 지식들.

- 수학과 형이상학이 종합적이든 분석적이든 필연적이라는 데는 모두 동의함.
- 수학은 발전을 거듭했으나, 형이상학은 발전하지 않았다라고 생각함.
  - 이성은 자기가 경함한 것을 확장해서 전체에 대해서 생각할 수 있음. 경험을 바탕으로 아직 경험하지 않은 것에 적용할 수 있는 것이 이성 능력이고, 그 결과물이 형이상학이다.
  - ex) 신이 존재한다, 영원이 있다
  - 이런 차이를 만든 원인은 무엇이고, 형이상학도 진정한 선험적 학문이 될 수 있는지 따져 보려고 함.
- 흄 덕분에 독단의 잠에서 깨어남
  - 흄 : 경험 독립적으로 세계 전체에 대해서 주장을 해도 우리는 사물 자체와 비교할 수 없고 확인할 수 없다는 회의론
  - 개연적으로만 알고 있다고 말할 수 있고, 필연적인 세계에 대한 필연적 지식은 불가능하다고 주장함.
- 하지만 흄과 다르게 세계에 대한 선험적이고 객관적인 인식이 여전히 가능함을 증명하려고 함.

### 선험적 종합 판단은 어떻게 가능한가?

선험적이고 종합적인 인식이 가능하지만, 인식의 범위를 엄격하게 제한해야 함

- 감각 지각할 수 있는 대상에 관한 한, 선험적 종합 판단은 가능함
  - 흄 : 인과율의 필연성을 부정
  - 칸트 : 현상, 드러나는 것에 대해서 판단할 때에는 언제든지 인과율을 적용할 수밖에 없다. 우리 인식 구조 자체가 그렇게 정해져 있다.
  - ex) 어떤 사건이 발생하면 우리 인식 구조가 원인을 찾도록 돌아간다는 것이다.
- 감각 지각할 수 없는 신이나 영혼, 자유 의지, 세계 전체, 사물 전체에 오성의 초월적 범주를 적용하려고 시도할 때 이율배반에 빠짐
  - 이것은 오용이고, 월권임.
  - 우리의 선험적, 초월적 범주는 가능한 경험 자료가 주어졌을 때만 적용할 수 있음. 그것을 넘어선 것에 범주를 적용하게 되면 모순에 빠짐.

⇒ 선험적인 범주, 선험적 원칙은 감각 대상에 적용해야 함.

### 선험적 원칙

선험적 원칙 : 경험이 가능하기 위해서 우리가 반드시 따라야 하는 규칙

ex) 인과율, 실체의 지속성

- 내재적(immanent) 사용 : 선험적 원칙을 가능한 감각 지각의 대상에 적용하는 것
- 초재적(transcendent) 사용 : 선험적 원칙을 가능한 감각 지각의 범위를 넘어서 적용하는 것
- 초재적과 초월적을 혼동해서는 안 됨.
  - 초재적 : 선험적 원칙의 오용
  - 초월적 : 현상계에 대한 경험과 인식을 가능하게 만드는 조건
  - 칸트의 순수 이성 비판이 초월 철학이고 초월 논증이다 → 현상계에 대한 경험과 인식을 가능하게 만드는 조건에 대한 탐구

순수이성비판의 과제 : 귀납을 통해 도출되지 않고 선험적으로 알려진 보편 명제(ex. 인과율, 실체의 지속성)를 경험 안에서 사용하는 것이 정당함을 밝히고, 가능한 감각 지각의 범위를 넘어선 영역, 사물 자체에 적용하면 정당성을 잃는다는 점을 보여주는 것

### 이율배반

1.  - 세계는 시간상 시초가 있으며 공간상 한계가 있다
    - 세계는 시간상 시초가 없으며 공간상 한계가 없다
2.  - 모든 합성 실체는 단순한 부분으로 구성되고, 따라서 모든 실체는 단순한 것이거나 단순한 부분을 갖는 합성된 것이다
    - 어떤 합성 실체도 단순한 부분으로 구성되지 않으며, 따라서 단순 실체는 없다

    단순한 것의 대표적인 예 : 영혼

    전자 = 전통 형이상학

    후자 = 유물론

3.  - 자연법칙에 따른 원인성은 모든 현상을 설명하기에 충분치 않고, 모든 현상을 설명하기 위해 자유의 원인성을 가정할 필요가 있다
    - 자유는 없으며 모든 현상은 자연법칙에 따라 발생한다

    전자 = 전통적인 형이상학

    후자 = 유물론

이론적으로 극복할 방법은 없다.

칸트는 실천적으로 극복한다.

## 인식의 가능 조건

인간의 인식에는 우리가 알지 못하지만 아마도 공통된 뿌리에서 솟아난 두 원천, 곧 감성과 오성(지성, 이해력)이 있음

- 감성 : 자극 정보를 수용하는 능력
- 오성 : 정보를 배열하는 능력

칸트에서는 넓은 의미의 이성과 좁은 의미의 이성이 있음

- 넓은 의미의 이성 : 감성과 오성 능력을 포함함
- 좁은 의미의 이성 : 사물 전체에 대해서 추론하고 생각하는 능력.

감성으로 사물에서 비롯한 표상을 얻음.

> 직관이 없는 사고는 공허하며, 개념이 없는 직관은 행복이다

### 통각

감성으로 직관한 다음 다양한 표상들이 개념을 매개로 어떤 대상에 속한 것으로 통일될 때만 우리는 어떤 대상을 인식함

우리가 공통적으로 가진 감성 능력, 오성 능력, 통각 능력이 바탕에 깔려서 현상을 인식할 수 있음.

- 주체의 인식 조건이 동시에 경험되는 대상을 가능하게 만든다
- 경험되는 사물을 인식할 수 있는 근거가 세계에 현실적으로 있는 사물이 가능한 근거
- 우리의 인식 조건을 떠나서, 인식 조건과 무관하게 사물 자체를 알 길은 없다

따라서 칸트의 인식론은 현상 존재론

- 현상계는 정신 독립적으로 객관적으로 있다
- 현상에 대한 객관적 인식은 가능하다
